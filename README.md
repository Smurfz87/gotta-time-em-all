# gotta-time-em-all

A mobile-first sports timing app for coaches and trainers. Record splits for multiple participants simultaneously in Heat mode (one finish time per participant) or Lap mode (continuous splits per participant).

## Usage

Live at: https://smurfz87.github.io/gotta-time-em-all/

## Development

```bash
npm install
npm run dev
```
